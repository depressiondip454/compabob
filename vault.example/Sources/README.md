# Sources

Imported and reference material: documents, articles, transcripts, exports, anything brought in from outside.

Keep one note per source. Give each a frontmatter `source:` field (where it came from) and a `date:`. Attachments go in `Sources/attachments/`.

This keeps reference material separate from {{USER_NAME}}'s own thinking, which lives in the other folders.
